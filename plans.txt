# основные цели бота
---------ядро---------
---->>поиск
    - есть протеин --> по нему находится таблица мотивов
    - ELM -- eukariotic linear motif -- единица(строка) в таблице 
    - по нему нужно взять Interaction Domain(домен взаимодействия)
    - из этого Interaction Domain нужно взять все протеины(они могут взаимодействовать с начальным протеином по этому домайну)
    - для каждого такого протеина нужно найти все гены

    в конце поиска известно:
        - имя начального протеина
        - имена ELM для каждого ELM есть
            + имена домайнов для каждого Interaction Domain есть
                            * имена протеинов из этих домайнов вместе с генами
